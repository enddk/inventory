<h2>モノ管理アプリ</h2> 
laravelとvueを使って、モノを管理するアプリを作りました。<br>
awsでバックエンドを、firebaseでフロントエンドを作成しました。<br>

<h2>機能</h2>
ユーザー機能（ログインしないと使えません）
投稿機能、画像投稿機能
入力情報のバリデーション
一覧表示の検索機能
vue-routerのルーディング管理

<h2>使用した言語</h2>
php8.1
laravel9.0
Node16.14.0
npm8.3.1
vue.js5.0.1

firebase
aws(vpc, ec2, rds, s3, route53)
mysql8.1.2
